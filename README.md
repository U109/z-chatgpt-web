# chatgpt-web-vue3

chatgpt-web-vue3使用vue3的脚手架创建开发，使用使用typescript编写代码。

## 项目介绍

chatgpt-web-vue3是web端，后端服务移步[z-chatgpt-data](https://gitee.com/zhangzhongzhen/z-chatgpt-data)

目前正在处于开发阶段，ts第一次使用，代码写的比较粗糙，后期会慢慢优化。

## 分支结构
- master：主分支，也是稳定分支
-  dev：开发分支，用于最新代码的开发


## 项目运行
```
npm install

npm run serve
```
